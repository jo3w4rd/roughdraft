    Class AudioImporter

    Audio importer lets you modify AudioClip import settings from editor scripts.

    Inheritance

    defaultSampleSettings

    The default sample settings for the AudioClip importer.

    Declaration

    Property Value
    Type Description
    AudioImporterSampleSettings
    Remarks

    These are the default settings for the AudioClip. If there are no overrides defined for a given build target, these settings will be used as the baseline for that platform.

    Examples
    
using UnityEngine;
using UnityEditor;

public class Example : AssetPostprocessor
{
    void OnPreprocessAudio()
    {
        AudioImporter audioImporter = assetImporter as AudioImporter;
        if (assetPath.Contains("mono"))
        {
            audioImporter = assetImporter as AudioImporter;
            audioImporter.forceToMono = true;
        }
    }
}

    preloadAudioData

    Preloads audio data of the clip when the clip asset is loaded. When this flag is off, scripts have to call AudioClip.LoadAudioData() to load the data before the clip can be played. Properties like length, channels and format are available before the audio data has been loaded.
