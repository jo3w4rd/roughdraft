    AssetImporterEditor

    Default editor for all asset importer settings.

    AssetImporterEditor.ExtraDataSerializedObject

    This class allows us to save the dirty state of the current editor targets. We save it on each ApplyModifiedProperties (at the end of the Inspector GUI loop) If the dirty count changed during the Update (at the beginning of the Inspector GUI loop) That means the target has been updated outside of the editor (by calling Reset, applying a Preset or performing any static menu action) And thus we need to re-initialize the extra instances before updating the serializedObject.
