    Class ClutchShortcutAttribute

    Attribute used to register a static method as the action for a ''clutch shortcut''.

    Remarks

    A clutch shortcut is triggered when the binding for the shortcut is pressed down and also when it is released. The method on which this attribute is placed must take a single argument of type ShortcutManagement.ShortcutArguments.
