    Struct ShortcutArguments

    Provides data for shortcut action methods invoked by the shortcut system.

    Namespace: UnityEditor.ShortcutManagement
    Syntax
    public struct ShortcutArguments
    Remarks

    See ShortcutManagement.ShortcutAttribute and ShortcutManagement.ClutchShortcutAttribute.

    Fields

    context

    Instance of the context in which the shortcut was triggered.

    Declaration
    public object context
    Field Value
    Type Description
    Object
    Remarks

    If the triggered shortcut was registered with a context of type EditorWindow then @@context@@ will be the instance of the EditorWindow in which the shortcut was triggered.

    stage

    The stage at which a shortcut action was invoked.

    Declaration
    public ShortcutStage stage
    Field Value
    Type Description
    ShortcutStage
    Remarks

    This is used to determine if the action of a clutch shortcut was invoked due to pressing the binding (ShortcutManagement.ShortcutState.Begin) or releasing the binding (ShortcutManagement.ShortcutState.End).
