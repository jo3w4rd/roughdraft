    Enum ComputeBufferType

    ComputeBuffer type.

    Namespace: UnityEngine
    Syntax
    public enum ComputeBufferType
    Remarks

    Different types of compute buffers map to different usage and declarations in HLSL shaders. Default type is "structured buffer" (@@StructuredBuffer@@ or @@RWStructuredBuffer@@).

    SA: ComputeBuffer, ComputeShader, Material.SetBuffer.

    Fields

    Append-consume ComputeBuffer type.

    Structured
