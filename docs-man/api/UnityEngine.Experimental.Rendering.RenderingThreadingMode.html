    Enum RenderingThreadingMode

    Enumeration describing the application's actual rendering threading mode.

    Namespace: UnityEngine.Experimental.Rendering
    Syntax
    public enum RenderingThreadingMode
    Remarks

    Rendering threading mode is decided during the Unity application startup, according to the combination of Player Settings PlayerSettings::ref::MTRendering, PlayerSettings::ref::graphicsJobs and PlayerSettings::ref::graphicsJobsMode, as well as according to the target platform capabilities.

    It can later be queried using the property GraphicsDeviceSettings::ref::renderingThreadingMode.

    A comparison of different rendering threading modes can be found in tutorial page Multithreaded Rendering & Graphics Jobs .

    Fields

    Name Description
    Direct

    Direct rendering from main thread.
    LegacyJobified

    Several worker threads generate intermediate graphics commands. A single render thread converts them into low-level platform API graphics commands.
    MultiThreaded

    Main thread generates intermediate graphics commands. Render thread converts them into low-level platform API graphics commands.
    NativeGraphicsJobs

    Main thread generates intermediate graphics commands. Render thread converts them into low-level platform API graphics commands. Render thread can also dispatch graphics jobs to several worker threads.
    NativeGraphicsJobsWithoutRenderThread

    Several worker threads generate intermediate graphics commands and convert them into low-level platform API graphics commands.
    SingleThreaded

    Rendering threading mode used for internal debugging. Simulates RenderingThreadingMode::ref::MultiThreaded but uses only a single thread.
