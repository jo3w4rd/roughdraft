    Class VFXEventAttribute

    The class VFXEventAttribute handles properties transmitted to a system with a Experimental.VFX.VisualEffect.SendEvent.

    Inheritance
    Object
    VFXEventAttribute
    Namespace: UnityEngine.Experimental.VFX
    Syntax
    public sealed class VFXEventAttribute : IDisposable
    Remarks

    Lets you alter initial properties from a Spawn.
