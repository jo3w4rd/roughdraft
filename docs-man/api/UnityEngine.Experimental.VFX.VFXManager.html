    Class VFXManager

    The VFX Manager lets you set a number of properties that control VisualEffect behavior within your game.

    Properties

    fixedTimeStep

    The fixed interval in which the frame rate updates. The tick rate is in seconds.

    Declaration
    public static float fixedTimeStep { get; set; }
    Property Value
    Type Description
    Single
    Remarks

    For in-game time, the /fixedDeltaTime/ and /deltaTime/ intervals are affected by Time.timeScale. These values are less than or equal to Experimental.VFX.VFXManager.maxDeltaTime.

    The /fixedDeltaTime/ interval can be an integer multiple of Experimental.VFX.VFXManager.fixedTimeStep.

    maxDeltaTime

    The maximum allowed delta time for an update interval. This limit affects /fixedDeltaTime/ and /deltaTime/. The tick rate is in seconds.
