    Struct MeshInfo

    Contains state information related to a tracked mesh.

    Namespace: UnityEngine.Experimental.XR
    Syntax
    public struct MeshInfo : IEquatable<MeshInfo>
    Remarks
                This struct is used by the [XRMeshSubsystem](xref:UnityEngine.Experimental.XR.XRMeshSubsystem) to determine which meshes have been added, updated, or removed.

            SA: [XRMeshSubsystem.TryGetMeshInfos](xref:UnityEngine.Experimental.XR.XRMeshSubsystem.TryGetMeshInfos)

    Properties

    ChangeState

    The change state (e.g., Added, Removed) of the tracked mesh.

    Declaration
    public MeshChangeState ChangeState { get; set; }
    Property Value
    Type Description
    MeshChangeState
    Remarks
                    The /ChangeState/ is relative to the last call to [XRMeshSubsystem.TryGetMeshInfos](xref:UnityEngine.Experimental.XR.XRMeshSubsystem.TryGetMeshInfos). For example, a value of [MeshChangeState.Added](xref:UnityEngine.Experimental.XR.MeshChangeState.Added) indicates the mesh has been added since the last call to /TryGetMeshInfos/.
