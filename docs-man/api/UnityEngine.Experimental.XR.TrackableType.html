    Enum TrackableType

    A trackable is feature in the physical environment that a device is able to track, such as a plane.

    Namespace: UnityEngine.Experimental.XR
    Syntax
    public enum TrackableType
    Remarks
                /TrackableType/ may be used as a flag and passed to methods like [XRRaycastSubsystem.Raycast](xref:UnityEngine.Experimental.XR.XRRaycastSubsystem.Raycast) to specify different
            types of trackables to raycast against.
