    Class Gizmos

    Gizmos are used to give visual debugging or setup aids in the Scene view.

    Inheritance
    Object
    Gizmos
    Namespace: UnityEngine
    Syntax
    public sealed class Gizmos
    Remarks

    All gizmo drawing has to be done in either MonoBehaviour::pref::OnDrawGizmos or MonoBehaviour::pref::OnDrawGizmosSelected functions of the script.

    MonoBehaviour::pref::OnDrawGizmos is called every frame. All gizmos rendered within MonoBehaviour::pref::OnDrawGizmos are pickable. MonoBehaviour::pref::OnDrawGizmosSelected is called only if the object the script is attached to is selected.

    Properties

    color

    Sets the color for the gizmos that will be drawn next.

    Declaration
    public static Color color { get; set; }
    Property Value
    Type Description
    Color
    Remarks

    Any color can be applied to gizmos. The example below shows the use of a red ::ref::color.

    Examples
    
using UnityEngine;
using System.Collections;

public class ExampleClass : MonoBehaviour
{
    void OnDrawGizmosSelected()
    {
        // Draws a 5 unit long red line in front of the object
        Gizmos.color = Color.red;
        Vector3 direction = transform.TransformDirection(Vector3.forward) * 5;
        Gizmos.DrawRay(transform.position, direction);
    }
}
