    Struct HumanLimit

    This class stores the rotation limits that define the ''muscle'' for a single human bone.

    center

    The default orientation of a bone when no muscle action is applied.

    Declaration
    public Vector3 center { get; set; }
    Property Value
    Type Description
    Vector3
    Remarks

    The vector value represents the bone's rotation in degrees around the X, Y and Z axes relative to the initial position of the bone in the skeleton. Any muscle rotation subsequently applied to the bone will be relative to this value.

    If ::ref::useDefaultValues is enabled, the value of /center/ will be [0, 0, 0].

    SA: HumanLimit.useDefaultValues.

    max

    The maximum rotation away from the initial value that this muscle can apply.

    Declaration
    public Vector3 max { get; set; }
    Property Value
    Type Description
    Vector3
    Remarks

    The ::ref::center property specifies the rotation of the bone when the muscle is at "rest". The /max/ value specfies the maximum rotation in degrees away from the rest value that the muscle can apply. The ::ref::min value specifies a similar limit but in the negative direction of rotation.

    The allowed range for the maximum is 0..180 degrees.

    SA: HumanLimit.useDefaultValues.
