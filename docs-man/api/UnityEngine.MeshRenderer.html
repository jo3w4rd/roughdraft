    Properties

    additionalVertexStreams

    Vertex attributes in this mesh will override or add attributes of the primary mesh in the MeshRenderer.

    Declaration
    public Mesh additionalVertexStreams { get; set; }
    Property Value
    Type Description
    Mesh
    Remarks

    This is used for the UVs for realtime lightmaps, but could also be used for vertex painting tools, etc.

        For example, if the primary mesh doesn't contain the [Mesh.colors](xref:UnityEngine.Mesh.colors) channel, the colors from additionalVertexStreams are used. If the primary mesh contains the colors channel and the additionalVertexStreams mesh also has colors, then colors from additionalVertexStreams are used.

    subMeshStartIndex

    Remarks
