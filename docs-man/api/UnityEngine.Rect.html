    Struct Rect

    A 2D Rectangle defined by X and Y position, width and height.

    Namespace: UnityEngine
    Syntax
    public struct Rect : IEquatable<Rect>
    Remarks

    Unity uses a number of 2D coordinate spaces, most of which define X as increasing to the right, and Y increasing upwards. The one exception is in the GUI and GUILayout classes, where Y increases downwards.

    The following examples are illustrated in GUI space, where (0,0) represents the top-left corner and Y increases downwards.

    Rectangles can be specified in two different ways. The first is with an ::ref::x and ::ref::y position and a ::ref::width and ::ref::height:

    {img RectXY.svg}

    The other way is with the X and Y coordinates of each of its edges. These are called ::ref::xMin, ::ref::xMax, ::ref::yMin and ::ref::yMax:

    {img RectXMinYMin.svg}

    Note that although /x/ and /y/ have the same values as /xMin/ and /yMin/, they behave differently when you set them. Setting /x/ or /y/ changes the position of the rectangle, but preserves its size:

    {img RectSetX.svg}

    Setting any of /xMin/, /xMax/, /yMin/ and /yMax/ will resize the rectangle, but preserve the position of the opposite edge:

    {img RectSetXMin.svg}

    SA: [[wiki:gui-Basics|GUI Scripting Guide]], Camera.rect, Camera.pixelRect.

    Remarks

    This is the same as ::ref::min, except that setting it will move the rectangle rather than resize it.

    Examples
    
using UnityEngine;

// Show Rect, the position selected randomly.

public class RectExample : MonoBehaviour
{
    private Rect rect;

    void Start()
    {
        rect = new Rect(600 * Random.value, 450 * Random.value, 200, 150);
    }

    void Update()
    {
        if (Input.GetKeyDown("space"))
        {
            rect.position = new Vector2(600 * Random.value, 450 * Random.value);
        }
    }

    // display a rectangle
    void OnGUI()
    {
        GUI.Box(rect, "");
    }
}
