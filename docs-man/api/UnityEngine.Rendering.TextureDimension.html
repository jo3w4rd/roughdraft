    Enum TextureDimension

    Texture "dimension" (type).

    Namespace: UnityEngine.Rendering
    Syntax

    Remarks

    Indicates type of a texture (2D texture, cubemap, 3D volume texture etc.).

    SA: Texture.dimension, MaterialProperty.textureDimension.
