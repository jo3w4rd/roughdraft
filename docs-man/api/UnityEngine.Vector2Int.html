    Struct Vector2Int

    Representation of 2D vectors and points using integers.

    Namespace: UnityEngine
    Syntax
    public struct Vector2Int : IEquatable<Vector2Int>
    Remarks

    This structure is used in some places to represent 2D positions and vectors that don't require the precision of floating-point.

    Property Value
    Type Description
    Single
    Remarks

    The length of the vector is square root of @@(xx+yy)@@.

    If you only need to compare magnitudes of some vectors, you can compare squared magnitudes of them using ::ref::sqrMagnitude (computing squared magnitudes is faster).

    SA: ::ref::sqrMagnitude.
