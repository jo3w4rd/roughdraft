    Struct Vector4

    Representation of four-dimensional vectors.

    Namespace: UnityEngine
    Syntax
    public struct Vector4 : IEquatable<Vector4>
    Remarks

    This structure is used in some places to represent four component vectors (e.g. mesh tangents, parameters for shaders). In the majority of other cases a Vector3 is used.

    magnitude

    Returns the length of this vector (RO).

    Declaration
    public float magnitude { get; }
    Property Value
    Type Description
    Single
    Remarks

    The length of the vector is square root of @@(xx+yy+zz+ww)@@.

    If you only need to compare magnitudes of some vectors, you can compare squared magnitudes of them using ::ref::sqrMagnitude (computing squared magnitudes is faster).

    SA: ::ref::sqrMagnitude.

    Property Value
    Type Description
    Vector4
    Remarks

    Note that the current vector is unchanged and a new normalized vector is returned. If you want to normalize the current vector, use ::ref::Normalize function.

    If the vector is too small to be normalized a zero vector will be returned.

    SA: ::ref::Normalize function.
