    Struct InputDevice

    Defines an input device in the XR input subsystem.

    Namespace: UnityEngine.XR
    Syntax
    public struct InputDevice : IEquatable<InputDevice>
    Remarks

    To retrieve input features or route haptic feedback to XR input devices, specify an XR.XRNode as the destination. Use XR.XRNode.LeftHand and XR.XRNode.RightHand to send haptic data to left or right devices. You can send haptic data either as an impulse or as a buffer of raw bytes that is played back through the haptic device. You can stop haptic output or query the device for its buffered capabilities at any time.

    Properties

    characteristics

    Read Only. A bitmask of enumerated flags describing the characteristics of this InputDevice.

    Declaration
    public InputDeviceCharacteristics characteristics { get; }
    Property Value
    Type Description
    InputDeviceCharacteristics
    Remarks

    Use Characteristics to determine whether an InputDevice has specific features or capabilities. For example, if the set of XR.InputDeviceCharacteristics includes both XR.InputDeviceCharacteristics::pref::Hand and XR.InputDeviceCharacteristics::pref::Left, then the device provides tracking data for the left hand.

    Characteristics is a bitmask. Use the bitwise operators to test for specific flags. For example, to determine whether an InputDevice has a camera, use the conditional:

    @@(inputDevice.characteristics & InputDeviceFlags.Camera) == InputDeviceFlags.Camera@@.
