    Class InputDevices

    An interface for accessing devices in the XR input subsytem.

    Inheritance
    Object
    InputDevices
    Namespace: UnityEngine.XR
    Syntax
    public class InputDevices
    Remarks

    To route haptic feedback to XR input devices, specify an XR.XRNode as the destination. This interface provides access to input devices using an XRNode. For example, use the use XR.XRNode.LeftHand and XR.XRNode.RightHand to access the left or right devices.
