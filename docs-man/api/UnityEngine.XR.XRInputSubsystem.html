    Class XRInputSubsystem

    XRInputSubsystem Instance is used to enable and disable the inputs coming from a specific plugin.

    Syntax
    public class XRInputSubsystem : IntegratedSubsystem<XRInputSubsystemDescriptor>, ISubsystem
    Remarks

    Starting up an XRInputSubsystem Instance will cause that plugin to start feeding input device data to the following XR.InputTracking systems: XR.InputTracking.GetLocalPosition and XR.InputTracking.GetLocalRotation. Calling the Stop or Shutdown functions will disable polling any input device data for that plugin.
