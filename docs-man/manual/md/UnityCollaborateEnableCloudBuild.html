    Enabling Cloud Build with Collaborate

    Unity Cloud Build is a continuous integration service for Unity Projects. For more information, see Unity Cloud Build.

    Before you can use Cloud Build with Collaborate, you must enable Collaborate on your Unity Project. For more information, see Setting up Unity Collaborate.

    To enable Cloud Build with Unity Collaborate:

    1. On the Unity Editor menu bar, select Window > General > Services.
    2. In the Services window, click the Collaborate tab.
    3. In the Collaborate window, click Open the history panel.

    4. To open the Cloud Build window, click the build now button.

    5. To enable Cloud Build, click the Build games faster toggle.

    6. From the PLATFORM drop-down menu, select the build platform.

    7. Click Next.

    8. In the TARGET SETUP window:

      1. In the TARGET LABEL textbox, enter a name to identify the build.
      2. From the UNITY VERSION drop-down menu, choose the Unity version to use for the build.

      3. If your root folder does not contain your Assets and Project Settings, specify the location in the PROJECT SUBFOLDER textbox.

    9. To start the initial build of your Project, click Next: Build.
