    Google VR controllers and input devices

    Currently the only official, first-party input device for Daydream is the Daydream controller. Google Cardboard applications can use a variety of third party controllers to take input.

    Daydream controller

    The Daydream controller allows 3 degrees of freedom, providing rotational and positional information, a dual axis touch/click controller along with 2 extra buttons.

    For information on how to take input from this device see the Google Daydream Controller documentation.
