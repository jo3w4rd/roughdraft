    Importing and activating the Target Database in Unity

    In the Unity Editor, navigate to Asset > Import Package > Custom Package and find the package on your hard drive. In the Import Unity Package window, click the Import button.

    Select the ARCamera from the Hierarchy window and, in the Inspector window, navigate to the Vuforia Behaviour (Script) component and click on the Open Vuforia configuration button.

    In the Vuforia Configuration window, under Datasets, select and check both the Load [DatabaseName] Database and Activate checkboxes. This activates your Image Target database for use with Unity.
