    Struct BoundsInt

    Represents an axis aligned bounding box with all values as integers.

    Namespace: UnityEngine
    Syntax
    public struct BoundsInt : IEquatable<BoundsInt>
    Remarks

    An axis-aligned bounding box, or AABB for short, is a box aligned with coordinate axes and fully enclosing some object. As the box is never rotated with respect to the axes, it can be defined by ::ref::min and ::ref::max points.
