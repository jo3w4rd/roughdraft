    Struct ColliderDistance2D

    Represents the separation or overlap of two Collider2D.

    Namespace: UnityEngine
    Syntax
    public struct ColliderDistance2D
    Remarks

    The ColliderDistance2D primarily defines a point on the exterior of each Collider2D along with the ::ref::distance between those two points. The ::ref::distance between them can be positive indicating that the Collider2D are separated (not overlapped), zero indicating that they are touching (but not overlapped) or negative indicating that they are overlapped.

    A ::ref::normal is provided that is a normalized vector that points from ::ref::pointB to ::ref::pointA. This vector, when scaled with the ::ref::distance, provide a vector that can be used to move the Collider2D so that they are no longer overlapped (if the ::ref::distance is negative) or so they are touching (if the ::ref::distance is positive).

    A common use-case for this is solving overlaps between two Collider2D, particularly when attached to a Rigidbody2D set to be RigidbodyType2D.Kinematic.

    SA: Physics2D.Distance, Collider2D.Distance and Rigidbody2D.Distance.

    Remarks

    The distance between two colliders is defined as the length of a Vector2 between ::ref::pointA and ::ref::pointB. The ::ref::distance between these points can be positive indicating that the Collider2D are separated (not overlapped), zero indicating that they are touching (but not overlapped) or negative indicating that they are overlapped.

    Remarks

    The ::ref::distance can be positive indicating that the Collider2D are separated (not overlapped), zero indicating that they are touching (but not overlapped) or negative indicating that they are overlapped.

    This property is therefore a check to see if ::ref::distance is negative which indicates the Collider2D are overlapped.

    isValid

    Gets whether the distance is valid or not.

    Declaration
    public bool isValid { get; set; }
    Property Value
    Type Description
    Boolean
    Remarks

    When a function returns a ColliderDistance2D struct, it may not be valid because the Collider2D used as distance references were not valid in some way due to them being disabled or not having any valid shapes. This property indicates whether the other distance properties can be used and are valid or not.
