    Struct BoundedPlane

    Structure describing a bounded plane representing a real-world surface.

    Namespace: UnityEngine.Experimental.XR
    Syntax
    public struct BoundedPlane
    Remarks
                Unlike a [Plane](xref:UnityEngine.Experimental.XR.Plane), which is infinite, a [BoundedPlane](xref:UnityEngine.Experimental.XR.BoundedPlane) has a size and a more detailed shape.
            A [BoundedPlane](xref:UnityEngine.Experimental.XR.BoundedPlane) can change from frame to frame (see [XRPlaneSubsystem.PlaneUpdated](xref:UnityEngine.Experimental.XR.XRPlaneSubsystem.PlaneUpdated)). Typically, this
            means the current understanding of the plane has changed, rather than the real-world geometry. This can
            happen as the device explores the physical space and is able to perceive more detail.

    Remarks
                    The position of the plane generally stays the same between frames, unless the
                understanding of where the plane is changes. For example, the device's plane
                detection algorithm may determine that the surface associated with the plane
                is actualy lower than it originally thought. In this case, the position's Y
                value might change. But if the dimensions of the plane change as the surface
                is explored, the position will likely stay the same. This is different than
                the [Center](xref:UnityEngine.Experimental.XR.Center) which can change frequently.

                Thus, the /Pose.position/ provides a stable reference point to which you could make other [GameObject](xref:UnityEngine.Experimental.XR.GameObject)s relative.

                A rotation of identity means the plane is horizontal, with its normal pointing up
                (positive along the Y axis). The [BoundedPlane.Width](xref:UnityEngine.Experimental.XR.BoundedPlane.Width) corresponds to its size in the X direction, and its [BoundedPlane.Height](xref:UnityEngine.Experimental.XR.BoundedPlane.Height) corresponds to its size in the Z direction.

    SubsumedById

    A session-unique identifier for the BoundedPlane that subsumed this plane.

    Declaration
    public TrackableId SubsumedById { get; set; }
    Property Value
    Type Description
    TrackableId
    Remarks
                    If /SubsumedById/ is not [TrackableId.InvalidId](xref:UnityEngine.Experimental.XR.TrackableId.InvalidId), then this [BoundedPlane](xref:UnityEngine.Experimental.XR.BoundedPlane) has been merged into the [BoundedPlane](xref:UnityEngine.Experimental.XR.BoundedPlane) with the [TrackableId](xref:UnityEngine.Experimental.XR.TrackableId) specified by /SubsumedById/. This can happen, for example, when two parts of the same surface are initially detected as separate planes. As the physical space is explored, the device may determine that the two planes actually represent the same surface.

                Note: Not all platforms support this feature. Unity provides the information where available.

                SA: [BoundedPlane](xref:UnityEngine.Experimental.XR.BoundedPlane), [XRPlaneSubsystem](xref:UnityEngine.Experimental.XR.XRPlaneSubsystem), [XRPlaneSubsystem.TryGetPlane](xref:UnityEngine.Experimental.XR.XRPlaneSubsystem.TryGetPlane)
