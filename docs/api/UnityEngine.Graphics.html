    Class Graphics

    Raw interface to Unity's drawing functions.

    Inheritance
    Object
    Graphics
    Namespace: UnityEngine
    Syntax
    public class Graphics
    Remarks

    This is the high-level shortcut into the optimized mesh drawing functionality of Unity.
