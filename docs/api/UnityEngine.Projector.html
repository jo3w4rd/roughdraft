    Namespace: UnityEngine
    Syntax
    public sealed class Projector : Behaviour
    Remarks

    The Projector can be used to project any material onto the Scene - just like a real world projector. The properties exposed by this class are an exact match for the values in the Projector's inspector.

    It can be used to implement blob or projected shadows. You could also project an animated texture or a render texture that films another part of the Scene. The projector will render all objects in its view frustum with the provided material.

    There is no shortcut property in GameObject or Component to access the Projector, so you must use Component::pref::GetComponent to do it:

    Examples
    
using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        // Get the projector
        Projector proj = GetComponent<Projector>();
        // Use it
        proj.nearClipPlane = 0.5f;
    }
}

    Properties

    aspectRatio

    The aspect ratio of the projection.

    Declaration
    public float aspectRatio { get; set; }
    Property Value
    Type Description
    Single
    Remarks

    This is projection width divided by height. An aspect ratio of 1.0 makes the projection square; a ratio of 2.0 makes it twice as wide than high. SA: [[wiki:class-Projector|projector component]].

    Examples
    
using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        // Get the projector
        Projector proj = GetComponent<Projector>();
        // Use it
        proj.aspectRatio = 2.0f;
    }
}

    fieldOfView

    The field of view of the projection in degrees.

    Property Value

    Remarks

    This is the vertical field of view; horizontal FOV varies depending on the ::ref::aspectRatio. Field of view is ignored when projector is orthographic (see ::ref::orthographic). SA: [[wiki:class-Projector|projector component]].

    {

        // Use it

    }
}

    ignoreLayers

    Which object layers are ignored by the projector.

    Declaration
    public int ignoreLayers { get; set; }
    Property Value
    Type Description
    Int32
    Remarks

    See [[wiki:Layers|layer mask]].

    By default this is zero - i.e. no layers are ignored. Each set bit in /ignoreLayers/ will make this layer not affected by the projector.

    Examples
    
using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        Projector proj = GetComponent<Projector>();
        // Make the projector ignore Default (0) layer
        proj.ignoreLayers = (1 << 0);
    }
}

    material

    The material that will be projected onto every object.

    Declaration
    public Material material { get; set; }
    Property Value
    Type Description
    Material
    Remarks

    Note that unlike Renderer.material, this returns a shared material reference and not a unique duplicate.

    Projector does nothing if it has no material set up. The Standard Assets package contains an example of Projector's material in ''Blob-Shadow'' folder.

    SA: [[wiki:class-Projector|projector component]], Material class.

    orthographic

    Is the projection orthographic (''true'') or perspective (''false'')?

    Declaration
    public bool orthographic { get; set; }
    Property Value
    Type Description
    Boolean
    Remarks

    When orthographic is ''true'', projection is defined by ::ref::orthographicSize.

    When orthographic is ''false'', projection is defined by ::ref::fieldOfView. SA: [[wiki:class-Projector|projector component]].

    Examples
    
using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        Projector proj = GetComponent<Projector>();
        proj.orthographic = true;
    }
}

    orthographicSize

    Projection's half-size when in orthographic mode.

    Declaration
    public float orthographicSize { get; set; }
    Property Value
    Type Description
    Single
    Remarks

    This is half of the vertical size of the projection volume. Horizontal projection size varies depending on ::ref::aspectRatio. Orthographic size is ignored when projection is not orthographic (see ::ref::orthographic). SA: [[wiki:class-Projector|projector component]].

    Examples
    
using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        Projector proj = GetComponent<Projector>();
        proj.orthographic = true;
        proj.orthographicSize = 2.0f;
    }
}
