    Struct CullingResults

    Culling results (visible objects, lights, reflection probes).

    Namespace: UnityEngine.Rendering
    Syntax
    public struct CullingResults : IEquatable<CullingResults>
    Remarks

    In a scriptable render loop, the rendering process is typically doing culling for each camera (ScriptableRenderContext.Cull), followed by rendering subsets of visible objects (ScriptableRenderContext.DrawRenderers) and processing visible lights (::ref::visibleLights, ::ref::visibleReflectionProbes).

    CullingResults also provides several functions to aid shadow rendering (::ref::ComputeDirectionalShadowMatricesAndCullingPrimitives, ::ref::ComputeSpotShadowMatricesAndCullingPrimitives, ::ref::ComputePointShadowMatricesAndCullingPrimitives).
