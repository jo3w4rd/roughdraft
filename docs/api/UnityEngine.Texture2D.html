    Properties

    blackTexture

    Get a small texture with all black pixels.

    Declaration
    public static Texture2D blackTexture { get; }
    Property Value
    Type Description
    Texture2D
    Remarks

    All the RGBA colour values of this texture will be zero (0,0,0,0). (Note that the alpha is also zero, which is transparent).

    SA: ::ref::whiteTexture, ::ref::normalTexture.

    format

    The format of the pixel data in the texture (RO).

    Declaration
    public TextureFormat format { get; }
    Property Value
    Type Description
    TextureFormat
    Remarks

    Use this to determine the format of the texture.

    isReadable

    Returns true if the Read/Write Enabled checkbox was checked when the texture was imported; otherwise returns false. For a dynamic Texture created from script, always returns true. For additional information, see TextureImporter.isReadable.

    Declaration
    public override bool isReadable { get; }
    Property Value
    Type Description
    Boolean
    Overrides
    Texture.isReadable
    Remarks

    To use Texture2D.GetPixel, Texture2D.GetPixels and similar functions, the isReadable property must be set to true. If you call these functions on a Texture on which the the isReadable property is not set to true, an exception is thrown.

    normalTexture

    Gets a small texture with pixels that represent surface normal vectors at a neutral position.

    Declaration
    public static Texture2D normalTexture { get; }
    Property Value
    Type Description
    Texture2D
    Remarks

    All pixels of this texture are set to (0.5, 0.5, 1, 1).

    SA: ::ref::whiteTexture, ::ref::blackTexture.
