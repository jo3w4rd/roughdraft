    Struct Vector3Int

    Representation of 3D vectors and points using integers.

    Namespace: UnityEngine
    Syntax
    public struct Vector3Int : IEquatable<Vector3Int>
    Remarks

    This structure is used in some places to represent 3D positions and vectors that don't require the precision of floating-point.

    magnitude

    Returns the length of this vector (RO).

    Declaration
    public float magnitude { get; }
    Property Value
    Type Description
    Single
    Remarks

    The length of the vector is square root of @@(xx+yy+z*z)@@.

    If you only need to compare magnitudes of some vectors, you can compare squared magnitudes of them using ::ref::sqrMagnitude (computing squared magnitudes is faster).

    SA: ::ref::sqrMagnitude.
